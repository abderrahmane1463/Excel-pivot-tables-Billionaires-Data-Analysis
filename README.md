## Overview

A comprehensive Excel-based analysis of the Forbes Billionaires Statistics dataset. The workbook walks through the full data analysis lifecycle — from raw data cleaning to pivot-table driven insights and chart visualisations — answering key questions about how global wealth is distributed across industries, geographies, and age groups.

## Features

- Data cleaning and standardisation of the raw Forbes dataset
- Pivot table analysis covering industry, country, and age dimensions
- Interactive charts and visualisations summarising wealth concentration
- Descriptive statistics: mean, median, and distribution of net worth
- Structured insight and conclusion sheets summarising findings

## Tech Stack

| Category | Tools |
|----------|-------|
| Analysis Tool | Microsoft Excel |
| Techniques | Pivot Tables, Charts, Descriptive Statistics, Data Cleaning |
| Dataset | Forbes Billionaires Statistics (public dataset) |

## Project Structure

The workbook (`Billionaires_Analysis.xlsx`) is organised into dedicated sheets:

| Sheet | Content |
|-------|---------|
| Raw_Data | Original dataset as imported |
| Clean_Data | Cleaned and standardised data |
| Research_Questions | Analytical questions guiding the study |
| Pivot_Analysis | Pivot tables by industry, country, and age |
| Charts | Visual summaries of key findings |
| Statistics | Descriptive statistics |
| Insights | Interpreted findings |
| Conclusion | Executive summary |

## Results / Key Insights

- Wealth is highly concentrated in the technology and finance sectors, which together account for a disproportionate share of total billionaire net worth
- The United States and China dominate the billionaire landscape by count, though per-capita wealth concentration patterns differ significantly between the two
- The age distribution skews toward the 50–70 range, with self-made billionaires tending to be older on average than those who inherited their wealth
